# Changelog

All notable changes to BaseCheck Agent are documented here.

## [1.0.0-rc.1] - 2026-07-20

### Added

- Universal BaseCheck Agent with six bundled free control packs.
- Free checks for SQL Server, Oracle, PostgreSQL, Supabase (preview), and SQLite.
- Local JSON, generic HTTP/webhook, and syslog output.
- `--test-siem` output-delivery test and local SQLite demo.

### Scope

- Paid packs are distributed separately and require a valid entitlement.
- No native SIEM connectors are included.
