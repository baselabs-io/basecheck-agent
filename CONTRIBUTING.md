# Contributing to BaseCheck Agent

## Getting Started

```bash
git clone https://github.com/baselabs-io/basecheck-agent.git
cd basecheck-agent
go test ./...
go build ./cmd/agent
```

For major changes, open an issue first. Keep pull requests focused, add tests, and update documentation when behavior changes.

## Public Boundary

This repository contains the universal agent and bundled free packs only. Do not add paid or enterprise control-pack content, private backend-service code, credentials, internal documents, or customer data.
